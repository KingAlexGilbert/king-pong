# Early King Pong Builds

This folder contains early HTML prototypes that helped lead to King Pong.

These are not the current recommended versions of the game. They are preserved for project history so people can see how the idea started and how it began growing beyond basic Pong.

## Files

- `001-base-of-king-pong.html`  
  The first simple Pong-style arcade base that King Pong grew from.

- `002-inspiration-to-go-beyond.html`  
  The early difficulty version that added CPU difficulty options and helped inspire King Pong to grow beyond basic Pong.

## Why These Are Archived

King Pong did not begin as a complete game all at once. It started with a basic Pong-style prototype, then grew through many rounds of improvements, experiments, and platform conversions.

These early builds show the foundation of the project.
