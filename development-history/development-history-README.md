# King Pong Development History

This folder contains historical prompt-history documents and reconstructed making-of notes for King Pong.

These files show how King Pong evolved from a simple Pong-style browser game into a larger arcade campaign project with levels, save slots, music, controller support, touch support, Android packaging, Windows packaging, LAN play, and a secret boss.

## Files

- `prompt-history-001-beginnings.md`  
  Covers the earliest Pong-style prototype and the first steps toward expanding the idea.

- `prompt-history-002-making-the-html-game.md`  
  Covers the major HTML game development process, including campaign levels, menus, music, save slots, controller support, touch support, and the secret boss.

- `king-pong-android-apk-making-of.md`  
  Reconstructed making-of notes for the Android/APK conversion process.

- `king-pong-windows-desktop-making-of.md`  
  Reconstructed making-of notes for the Windows/Desktop conversion process.

## Note

Some files are prompt-history documents, while others are reconstructed making-of notes. Reconstructed documents are not perfect verbatim transcripts. They are included to preserve the project’s development story as accurately and honestly as possible.

Large code blocks, full generated files, and broken ChatGPT sandbox download links were intentionally removed or replaced with placeholders to keep the documents readable on GitHub.
