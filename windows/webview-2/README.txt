King Pong Desktop HD - WebView2 Fullscreen Build V11 ICON

Build instructions:
1. Install the .NET SDK and .NET Framework 4.8 Developer Pack if you have not already.
2. Run build-windows.bat.
3. Use the output folder: dist\KingPong-WebView2-Portable.
4. Share the whole KingPong-WebView2-Portable folder as a zip. Do not share only the EXE.

V11 changes:
- Adds the King Pong crown-ball icon to the Windows EXE.
- Sets the WinForms window/taskbar icon at runtime.
- Embeds the icon as the application icon so File Explorer and desktop shortcuts use it.
- Keeps the V10 in-game level picker and all previous fullscreen, DPI, autopause, no-flicker, and title-safety fixes.

Note:
If Windows still shows the old default icon after replacing a previous build, it may be using the Windows icon cache or an old desktop shortcut. Create a fresh shortcut to the new EXE, or restart Explorer/Windows.

V12 EXACT ICON:
- Uses the exact Android King Pong 512x512 crown-ball icon supplied by the project creator.
- Converted PNG to multi-size Windows .ico for taskbar, File Explorer, shortcuts, and Alt-Tab.
